### 本の紹介

- 鈴木 幸市、藤塚 勤也
- 翔泳社 (2005/2/22)
- 2017/5 現在廃盤
- 月刊DB Magazineの人気連載「RDBMS解剖学」を加筆/再編集し、書籍としてまとめたもの
